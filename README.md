## Quick Start

```bash
# Install
go install github.com/yourusername/wtx/cmd/wtx@latest

# Switch workspace (interactive)
wtx

# List all worktrees
wtx list

# Create new worktree
wtx add feature-branch

# Open in editor
wtx open feature-branch
```
